# Publicação automática no GitHub

Você não precisa passar senha para ninguém.

## Jeito mais automático no Windows

1. Extraia este ZIP.
2. Abra a pasta `dn-gamer-store`.
3. Clique duas vezes em `1_CLIQUE_PUBLICAR_NO_GITHUB.bat`.
4. Quando abrir o login oficial do GitHub, faça login no navegador.
5. No final, copie o link mostrado em `Site GitHub Pages` e cole no AVA.

O script tenta fazer tudo:

- instalar Git, caso esteja ausente;
- instalar GitHub CLI, caso esteja ausente;
- abrir login seguro do GitHub;
- criar repositório público;
- enviar todos os arquivos;
- ativar GitHub Pages na branch `main` e pasta `/`.

## Link esperado

Depois da publicação, o site deve ficar parecido com:

```text
https://SEU-USUARIO.github.io/dn-gamer-store/
```

## Segurança

Nunca envie senha, token, código 2FA ou código de recuperação em chats.
