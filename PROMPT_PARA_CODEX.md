# Prompt para usar no Codex

Use este prompt dentro da pasta do projeto `dn-gamer-store`:

```text
Revise este projeto front-end de e-commerce gamer para uma atividade avaliativa. Verifique se existem as páginas index.html e login.html, HTML5 semântico, CSS3, Bootstrap, responsividade com media queries, JavaScript e jQuery, menu, banner, destaque, novidades, galeria, rodapé, carousel, formulário de login e interações. Corrija bugs sem alterar a proposta visual, melhore a organização quando necessário e garanta que o site funcione abrindo o index.html no navegador. Depois, crie um resumo do que foi implementado e deixe o projeto pronto para publicar no GitHub Pages.
```
