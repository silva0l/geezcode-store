# DN Gamer Store — Projeto Front-End

Projeto desenvolvido para a atividade avaliativa de Front-End: criação das páginas **index** e **login** de um e-commerce de eletrônicos voltado ao universo gamer.

## O que foi desenvolvido

- Página inicial `index.html`
- Página de login `login.html`
- Menu responsivo com Bootstrap
- Banner principal com carousel
- Seção de destaques
- Seção de novidades
- Galeria com modal
- Formulário de newsletter com validação
- Formulário de login com validação
- Rodapé completo
- Layout responsivo com Bootstrap e media queries
- Interatividade com JavaScript e JQuery
- Uso de HTML5 semântico
- CSS3 com variáveis, efeitos visuais, cards, animações e responsividade
- Imagens locais em SVG dentro da pasta `assets/img`

## Tecnologias utilizadas

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- JavaScript
- JQuery
- LocalStorage

## Como abrir o projeto

1. Baixe ou extraia a pasta do projeto.
2. Abra o arquivo `index.html` no navegador.
3. Para testar o login, abra o arquivo `login.html` ou clique no botão **Login** do menu.

## Como publicar no GitHub

1. Crie uma conta ou acesse sua conta no GitHub.
2. Clique em **New repository**.
3. Nome sugerido: `dn-gamer-store`.
4. Marque o repositório como **Public**.
5. Envie todos os arquivos deste projeto para o repositório.
6. Depois, ative o GitHub Pages:
   - Vá em **Settings**.
   - Clique em **Pages**.
   - Em **Branch**, selecione `main` e `/root`.
   - Clique em **Save**.
7. Copie o link gerado pelo GitHub Pages e envie no AVA.
